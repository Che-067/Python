1. Functional vs Non-Functional Testing
Functional Testing - "Does it WORK?"

Tests WHAT the system does

Examples:

Login functionality

Payment processing

Search features

Focus: Requirements and specifications

Non-Functional Testing - "How WELL does it work?"

Tests HOW the system performs

Examples:

Speed (performance testing)

Security (security testing)

User experience (usability testing)

Focus: Quality attributes
